# kayron-cli

![npm版本](https://img.shields.io/badge/kayron-cli_v0.0.1-green)

## 📖 简介

- 用于快速搭建前端项目的命令行工具

## 📑 模板

- 快速生成 Vue3+Ts+Vite+ElementPlus+Pinia+Axios+Mock 的后台管理系统模板

## 🔑 安装

```bash
npm install kayron-cli -g
```

## 💡 使用

```bash
  # 使用模板创建新项目
  kayron create

  # 执行项目文件夹名称并选择模板创建新项目
  kayron create [projectName]

  # 更新脚手架到最新版本
  kayron update

  # 查看脚手架版本
  kayron -v
  kayron --version
```
